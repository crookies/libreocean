rpi3_xxx_defconfig files in this directory are created at different steps of the opencpn addition to the image in order to
identify which package were added at each step.

The post-image-std.sh is a copy of the original buildroot /board/raspberrypi3/post-image.sh file.
genimage-rpi3.cfg is a copy of the buildroot/board/raspberrypi3/genimage-raspberrypi3 file
